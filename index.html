This document serves as an official registry for all known global parameters, properties, and classes used by the Decentralized Identifier ecosystem.

This registry is under active development and implementers are advised against implementing the registry unless they are directly involved with the W3C DID Working Group.

Comments regarding this document are welcome. Please file issues directly on GitHub, or send them to public-did-wg@w3.org ( subscribe, archives).
